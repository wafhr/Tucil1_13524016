# Tugas Kecil 1 IF2211 Strategi Algoritma - Queens LinkedIn Solver

## Deskripsi Singkat
Program ini adalah aplikasi berbasis antarmuka grafis yang dibangun menggunakan Python untuk menyelesaikan teka-teki logika Queens. Pada permainan ini, pemain diminta menempatkan sejumlah Ratu di atas papan grid berukuran N x N yang telah terbagi ke dalam beberapa region warna. Aturan utama dalam menyelesaikan teka-teki ini adalah: setiap baris, kolom, dan wilayah warna hanya boleh ditempati oleh tepat satu Ratu. Selain itu, tidak boleh ada dua Ratu yang saling bersentuhan satu sama lain, baik secara vertikal, horizontal, maupun diagonal.

## Requirement & Instalasi
* Python 3.10+
* Package manager `pip` untuk menginstal dependensi (PyQt6, opencv-python, dan numpy)
* Jalankan perintah berikut di terminal dari direktori root proyek untuk menginstal seluruh dependensi:
  ```bash
  python -m pip install PyQt6 opencv-python numpy
  ```

## Cara Menjalankan Program
* Dari root proyek, jalankan perintah:
  ```bash
  python src/main.py
  ```
* Atau buka direktori `bin/` dan jalankan `QueensSolver.exe`
